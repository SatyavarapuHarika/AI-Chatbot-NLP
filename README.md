# 🤖 AI Chatbot (NLP + TF-IDF)

A web-based chatbot built using Natural Language Processing techniques like TF-IDF and cosine similarity.

---

## 🚀 Features

* NLP-based chatbot
* TF-IDF vectorization
* Cosine similarity matching
* Flask backend
* Modern chat UI

---

## 🛠 Tech Stack

* Python, Flask
* Scikit-learn
* HTML, CSS, JavaScript

---

## ⚙️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open: http://127.0.0.1:5000/
